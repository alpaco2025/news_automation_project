requests.txt 파일 설명

requests → 뉴스 사이트 접속해서 HTML 가져오기

openai → 요약/토픽/키워드 생성 LLM 호출

streamlit → 대시보드(UI)

pandas → 통계/집계/표 만들 때 편하게 쓰는 용도 (있으면 좋음)
