requests.txt 파일 설명
------한번에 로컬에서 우리 개발 관련된 라이브러리 한방에 설치 가능 ---------
requests : 웹 요청 (뉴스 사이트 HTML 가져오기)

beautifulsoup4 : HTML 파싱해서 제목/본문 추출

pymysql : Python → MySQL(RDS) 연결

python-dotenv : .env에서 DB 비번, OPENAI 키 등 읽기

openai : 요약/토픽/키워드 LLM 호출

fastapi : 웹 서버 / 라우팅 (URL → 함수)

uvicorn[standard] : FastAPI 앱을 띄우는 ASGI 서버

jinja2 : HTML 템플릿 엔진 (예쁜 웹페이지 디자인)

python-multipart : 나중에 form / 파일 업로드 다루고 싶을 때 필요 (미리 넣어두면 편함)

pandas : 나중에 통계/집계 보여줄 때 쓸 수 있음 (선택)
